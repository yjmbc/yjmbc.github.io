uniapp app 内点击返回 webview 退出前uni.showModal 弹框确认

背景

uniapp 做 app , app /a/ 路径下页面显示当前组件在 h5上的页面，以 webview 的形式放在页面上

当 app 点击返回的时候没有触发 webview 的页面回退，导致 webview 页面的路由守卫没有触发，弹框确认的逻辑放在路由守卫里，并且由 uni.showModal 来实现，在 app 环境下无法自定义样式，只能由 webview 触发路由守卫来实现。

解决方案

  1. app 内点击返回 webview 页面不回退

onReady 中获取 webview 实例并且判断当前页面是否可以回退

onBackPress 中针对是否可以回退做处理，可以回退直接回退，不可以回退做相关逻辑

参考文章： https://blog.15897.com/article/uniapp-webview-android-back

  1. 由于webview 页面栈只有当前页面一个，不可以回退，返回的时候不会触发路由守卫

在 onBackPress 中针对不可以回退的 webview 实例对象，利用 evalJS , 直接调起 uniapp 的 showModal 方法

相关代码

onReady() {
  // #ifdef APP
  var that = this
  var currentWebview = this.$scope
    .$getAppWebview() //此对象相当于html5plus里的plus.webview.currentWebview()。在uni-app里vue页面直接使用plus.webview.currentWebview()无效
  setTimeout(function() {
    that.wv = currentWebview.children()[0]
    that.onLoading()
  }, 1000); //如果是页面初始化调用时，需要延时一下
  // #endif
},
  onBackPress(options) {
    // 如果 webview 页面可以回退则让 webview 页面返回
    if (this.wvCanBack) {
      this.wv.back();
      return true;
    }
    // 不能返回则调起 uniapp 的 modal 弹框
    if (!this.wvCanBack) {
      var currentWebview = this.$scope
        .$getAppWebview()
      currentWebview.children()[0].evalJS(`this.wx.showModal({
					title: '确认退出？',
					success: function(res) {
						if (window.location.href.match(/from=([^&]+)/)?.[1] === 'app') {
							if (res.confirm) {
								uni.webView.postMessage({
									data: 'close'
								})
							}
							return
						}
					},
					cancelText: '取消',
					confirmText: '确认'
				})`)
      return true
    }
  },
  methods: {
    onLoading() {
      const that = this
      this.wv.onloading = function() {
        console.log('onloading')
        that.canBack()
      }
    },
    canBack() {
      const that = this
      that.wv.canBack(function(e) {
        if (e.canBack) {
          that.wvCanBack = 1
        } else {
          that.wvCanBack = 0
        }
      })
    },
    handleMessage(event) {
				if (event.detail.data && event.detail.data[0] === 'close') {
					uni.switchTab({
						url: '/pages/index/index',
					});
				}
			},
  },
  beforeRouteLeave(to, from, next) {
				uni.showModal({
					title: '确认退出？',
					success: function(res) {
							if (res.confirm) {
								uni.webView.postMessage({
									data: 'close'
								})
					},
					cancelText: '取消',
					confirmText: '确认'
				})
