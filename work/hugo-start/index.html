hugo 实践记录

安装

https://gohugo.io/installation/windows/直接用 scoop 安装最方便

go 安装msi文件运行就可以了

新建

  1. hugo new site [网站名字]

  2. 挑选一个主题 https://themes.gohugo.io/

  3. git submodule add [主题的github地址]

  4. 在 hugo.toml 里面定义主题 theme = '主题名字'

  5. hugo server 启动

  6. hugo server -D 预览

  7. 新建文章 hugo new content posts/my-first-post.md

  8. hugo 打包文件 会放到 public 目录下面

github page

  1. 创建一个仓库 命名 username.github.io
  2. 然后把远程仓库拉到本地 git clone 远程仓库
  3. 把上面 public 目录下的所有文件复制过来 提交
  4. 推送远程仓库
  5. 访问 username.github.io

hugo.toml 文件


# 要注意配置文件的 url 要更改成github page的路径！！！
baseURL = 'https://yjmbc.github.io/'

[Params]
# 指示网站是否为自托管
selfHosted = true

# 设置日期格式为"2006-01-02"，即年-月-日
dateFormat = "2006-01-02"

# 是否显示文章的预估阅读时间
readingTime = true

# 是否在文章末尾显示字数统计
wordCount = true

# 是否使用highlight.js进行代码高亮
useHLJS = true

# 是否在文章底部显示相关文章推荐
showRelatedPosts = true 

# 是否显示文章的最后修改时间
Lastmod = true

# 分页设置，每页显示的文章数量
pagination = "10"

# 是否在文章页面隐藏作者信息
hideAuthor = true

# 网站描述，可自定义内容
description="Your can add your website description here."

# 是否启用基于Lunr的客户端搜索功能
lunrSearch = true

# 预览卡片中图片的位置，此处设为显示在底部
previewCardImagePlacement = "bottom"

# 用来设置右上角的导航栏内容, url 对应 content 目录下的路径
[[menu.main]]
    name = "首页"
    url = "/"
    weight = 1

# weight 用来定义导航栏的展示顺序，数值越小越靠前
[[menu.main]]
    name = "学习记录"
    url = "/learn/"
    weight = 2

文章开头定义

使用主题： lightbi-hugo

---
title: hugo 实践记录
date: 2024-06-14
tags: ["hugo"]
image : "/img/work/bird.jpg"
Description  : "hugo 构建个人博客页并部署到 github page 全过程简便记录..."
---

tags 标签可以定义文章属于哪个标签，然后在 tags页面下按照标签展示img 是缩略卡片展示的图片内容以及内容页的展示图片，路径为themes/主题名字/assets/img下的路径
